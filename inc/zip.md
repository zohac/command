# Zip
## Install (Ubuntu)
```bash
apt-get install zip unzip
```

## zip
```bash
zip -r [filename].zip [filenameToZip]
```

## zip multiple directories in one named zip file
```bash
zip -r [filename].zip [filename1ToZip] [filename2ToZip] [filename3ToZip]
```

## unzip
```bash
unzip [filename].zip
```
